## Social Media Website centered around Art of the Institute of Chicago API

4 basic functions:
- Roles for Login
- Query Art, search 
- Favorite Art
- User profile creation

Stretch goals:

- User Functions

    - Logout
    - Register User

    - User has showcase Art in their profile

- Admin functions
    - Get list of users
    - Remove users

- Art API
    - Get Art attributes(API)
    - Get Art by Category(API)
    - View pictures(API)
